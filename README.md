# b3x - bear bear bear

Python script to automate tool installations in Termux

> Easy... Install hacking tools on your termux!!!

![](./screenshots/logo.png)

## Instalação

In terminal type:

```sh
git clone https://github.com/joaroquedev/b3x.git
```
```sh
python3 b3x.py
```
